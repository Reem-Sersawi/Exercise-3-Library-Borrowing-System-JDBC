## ما الذي قمت به من تعديلات؟

### 1. إكمال وظيفة زر `borrowedBooks`
- **الموقع:** ملف `BorrowController.java` و `BorrowDAO.java`
- **الوظيفة:** عرض جميع الكتب التي لا تزال مستعارة (التي لم تُرجع بعد)
- **آلية العمل:** يتم عرض السجلات التي يكون فيها الحقل `status = false`

### 2. إكمال وظيفة زر `search by ids`
- **الموقع:** ملف `BorrowController.java` و `BorrowDAO.java`
- **الوظيفة:** البحث عن سجلات الاستعارة باستخدام:
  - معرف الكتاب (Book ID) فقط
  - معرف الطالب (Student ID) فقط
  - كلا المعرفين معاً
- **آلية العمل:** يتم بناء استعلام SQL ديناميكي حسب المدخلات

### 3. إضافة حقول إدخال جديدة في واجهة المستخدم
- **الموقع:** ملف `Borrow.fxml`
- **الإضافات:**
  - حقل إدخال نصي للبحث بـ `Book ID`
  - حقل إدخال نصي للبحث بـ `Student ID`
  - إضافة فاصل (Separator) قبل قسم البحث

### 4. تحسين الكود الموجود
- **الموقع:** ملف `BorrowController.java`
- **التعديلات:**
  - إصلاح دالة `showInfoAlert` (كانت تستخدم نوع الخطأ WARNING بدلاً من INFORMATION)
  - إضافة التحقق من صحة المدخلات في البحث
  - إضافة معالجة أخطاء تحويل الأرقام (Number Format Exception)
  - تحسين رسائل التنبيه لتكون أكثر وضوحاً للمستخدم
